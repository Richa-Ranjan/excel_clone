# 📊 Excel Clone Web App

A lightweight and interactive Excel-like spreadsheet web application built using **HTML**, **CSS**, and **JavaScript**.

🔦 I’ve added a brand new **Dark Mode** feature to enhance user experience — especially for night owls!

---

## ✨ Features

- Dynamic cell editing and formatting
- Multiple sheet support
- Seamless UI interactions
- **Custom-built Dark Mode** toggle 🌙

---
